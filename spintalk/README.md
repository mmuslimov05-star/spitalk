# 🌀 SpinTalk — Случайный видеочат

> Полноценный аналог чат-рулетки с WebRTC, Socket.io, и панелью администратора.
> Оригинальный дизайн, собственная архитектура.

---

## 📁 Структура проекта

```
spintalk/
├── server.js              # Node.js бэкенд (Express + Socket.io)
├── package.json
├── public/
│   ├── index.html         # Главная страница (лендинг + чат)
│   ├── css/
│   │   └── style.css      # Все стили (тёмная/светлая тема)
│   └── js/
│       ├── app.js         # UI логика, интерфейс чата
│       └── webrtc.js      # WebRTC клиент (P2P видеозвонок)
└── admin/
    ├── login.html         # Страница входа
    ├── index.html         # Панель администратора
    ├── admin.css          # Стили admin-панели
    └── admin.js           # Логика admin-панели
```

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd spintalk
npm install
```

### 2. Настройка переменных окружения
```bash
# Создайте файл .env
PORT=3000
ADMIN_SECRET=your_secure_secret_here
ORIGIN=https://yourdomain.com
NODE_ENV=production
```

### 3. Запуск
```bash
# Разработка
npm run dev

# Продакшн
npm run prod

# С PM2 (автоперезапуск)
npm run pm2
```

### 4. Открыть в браузере
- Сайт: http://localhost:3000
- Админ вход: http://localhost:3000/admin/login.html

## ⚙️ Функциональность

### Пользовательская часть
- 📹 **Видеочат** — P2P видео через WebRTC
- 💬 **Текстчат** — только сообщения
- 🏷️ **Интересы** — фильтрация собеседников по тегам
- 🌍 **Фильтр по стране** — выбор страны собеседника
- ⏭ **Пропустить** — мгновенный переход к следующему
- 🚩 **Жалобы** — отправка жалоб на нарушителей
- 🎨 **Тёмная/светлая тема**
- 📷 **Управление камерой и микрофоном**

### Серверная часть
- 🔀 **Умный матчинг** — приоритет по общим интересам
- ⚡ **WebRTC сигналинг** — минимальная задержка
- 🤖 **Авто-бан** — при 3+ жалобах
- 🚫 **Фильтр стоп-слов** — в текстовом чате
- 🛡️ **Rate limiting** — защита от DDoS
- 📊 **Live статистика** — онлайн, чаты, очереди

### Панель администратора
| Раздел | Функции |
|--------|---------|
| 📊 Дашборд | KPI, графики активности, карта стран |
| 👥 Пользователи | Поиск, фильтрация, предупреждения, баны |
| 💬 Сессии | Список активных чатов, принудительное завершение |
| 🚩 Жалобы | Обработка, бан нарушителей, история |
| 🔨 Блокировки | IP-баны, временные/постоянные, разбан |
| 🛡️ Модерация | Авто-фильтры, стоп-слова, настройки AI |
| 📈 Аналитика | Часовая активность, страны, устройства |
| ⚙️ Настройки | Параметры сайта, модераторы, уведомления |
| 📋 Логи | Поток событий, экспорт CSV |

## 🌐 Деплой (пример с Nginx)

```nginx
server {
    listen 80;
    server_name yourdomain.com;
    
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_cache_bypass $http_upgrade;
    }
}
```
